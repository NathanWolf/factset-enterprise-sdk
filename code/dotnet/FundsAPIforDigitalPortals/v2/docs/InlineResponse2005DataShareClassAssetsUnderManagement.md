# FactSet.SDK.FundsAPIforDigitalPortals.Model.InlineResponse2005DataShareClassAssetsUnderManagement
Value range for the assets under management (AUM) which is the sum of the market values of the holdings of the share class including cash. The AUM is expressed in the currency of the share class, thus the values might be in different currencies if the parameter `shareClass.currency` has not been set unambiguously..
