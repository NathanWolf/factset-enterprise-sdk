# MetricMetric

Describes the entity's financial metrics category - FIN - Financial; Numerical value taken from an entity's financial statements. - RAT - Ratio; A relative magnitude of two or more selected numerical values taken from an entity's financial statements.  Exception: units = A (Actual) in combination with metric = RAT (Ratio) should be treated similarly as units = RATIO; the value is returned in the same format.
