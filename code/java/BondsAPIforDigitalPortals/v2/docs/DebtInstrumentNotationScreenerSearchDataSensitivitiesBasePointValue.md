

# DebtInstrumentNotationScreenerSearchDataSensitivitiesBasePointValue

Value range for the base point value, which is the change of the debt instrument's price for a one base point change (nominally, not relatively) of the prevailing market interest rate. One base point is equivalent to 0.01 percent (1/100 of a percent). The unit of the base point value is the value unit of the notation, thus the parameter `validation.valueUnit.restrict.ids` must be set to exactly one value unit in order to use the parameter.
